# Revenue Allocation & Treasury Compensation Model

This section explains **where money goes when Cosmic Friends earn** and how treasury funds can be allocated, including how founders/managers can be compensated transparently.

---

## üí∞ Where Does Money Go When Characters Earn?

When a Cosmic Friend earns from:
- livestream tipping  
- paid interactions  
- sponsorships  
- licensing  
- other on-chain income streams  

Revenue can be routed into **three possible buckets**, depending on the final design.

---

### Option A ‚Äî Everything Goes to the Main Treasury (MVP Recommended)

**Flow**  
Character Earnings ‚Üí COSMIC FRIENDS Treasury

**Benefits**
- Simple  
- Transparent  
- Ideal in early MVP  
- Treasury grows continuously  
- Governance has maximum power  
- No complexity in accounting  

**Trade-off**  
Characters don‚Äôt have individualized economic identity.

---

### Option B ‚Äî Split Between Treasury + Character Wallet

**Flow**  
Character Earnings ‚Üí
- % to Treasury  
- % to Character Sub-Treasury (per character)  

Each Cosmic Friend has their own wallet / sub-fund that can be governed by:
- the owner of that character  
- governance voting  
- or hybrid rules  

**Why this is cool**
- Characters become financially *independent beings*  
- Popular characters naturally gain more funding power  
- Creates economic storytelling  
- Increases emotional + financial attachment  

**Example Split**
- 60% Treasury  
- 30% Character Sub-Treasury  
- 10% ‚ÄúOps / Manager Fee‚Äù (explained below)  

Split is configurable via governance.

---

### Option C ‚Äî Everything Goes to Character Wallet (Experimental)

This is the **‚Äútrue AI sovereign being‚Äù** path.

Revenue Route:  
Character Earnings ‚Üí Character Wallet Only

Treasury earns indirectly through:
- mint auctions  
- royalties  
- ecosystem fees  

Philosophically interesting, but harder to sustain early.  
Not recommended for V1.
